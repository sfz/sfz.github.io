## Example

```
lfo02_phase=0.5
lfo01_phase=0.25
```

If the wave is a sine (the default shape), then phase 0 means the LFO starts
at value 0 (center) and goes up first. Phase 0.5 means starting at 0 and going
down first. Phase 0.25 means starting at 1 (max).
