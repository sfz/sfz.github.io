fileg_depth

Envelope depth.

For pitcheg and fileg, this is in cents, and can range from -12000 to 12000. -12000 cents is 10 octaves.

For ampeg, this should not normally be used.

Examples

fileg_depth=1200
pitcheg_depth=-100
Name Version Type Default Range Unit
fileg_depth SFZ v1 integer 0 -12000 to 12000 cents
Modulations
fileg_depth_onccN ARIA integer 0 -12000 to 12000 cents
fileg_depth_curveccN ARIA integer 0 0 to 255 N/A
fileg_vel2depth integer 0 -12000 to 12000 cents
